🐕 MetaWatch PRO

Вотчдог прямо во вкладке Ads Manager — резервный контур мониторинга, который работает даже когда серверная инфраструктура лежит. Сценарии по метрикам (свои + пресеты), тик по таймеру, стата по кабам, журнал сработок, опционально Telegram-алерты из браузера.

①
Перетащи кнопку ниже в свою панель закладок браузера (Bookmarks Bar).
🐕 MetaWatch PRO
②
Открой business.facebook.com или adsmanager.facebook.com (логин обязателен). Пользуйся всегда с одного домена — стейт хранится per-domain.
③
Нажми закладку MetaWatch PRO → панель справа → выбери кабы → повесь сценарии → ▶ СТАРТ. Вкладку запинь и не закрывай.

Что умеет

  • Сценарии-условия — конструктор «метрика / оператор / значение» (AND), уровень кампания или ад, фильтр по имени (напр. только БЕЗ CTRL)
  • Готовые пресеты — «Без кликов», «Дорогой CPC», «CTR умер», «Частота выжгла», K1/K4-lite, «CPM шторм», спенд-предохранитель
  • OBSERVE / ARMED — по умолчанию сценарий только пишет в журнал; тумблер ARMED (с подтверждением) включает реальную паузу. Авто-возврата из паузы нет и не будет
  • Тик по таймеру — 5/10/15/30/60 мин, worker-таймер переживает троттлинг фоновой вкладки, догоняющий тик после простоя
  • Стата по кабам — today: спенд / CTR / CPC / CPM / частота / реги / покупки по кампаниям, мультиаккаунт из одной вкладки
  • Журнал — каждая сработка с причиной и фактическими цифрами, экспорт JSON
  • Telegram-алерты — напрямую из браузера (без серверов), отдельный алерт-бот
  • Переживает reload — сценарии/кабы/журнал в localStorage: кликнул закладку — всё поднялось, вотчдог возобновился
  • FB-only метрики — трекер не нужен: работает на токене сессии, полная независимость от внешней инфры

Быстрый старт

🅐 Наблюдение (безопасно)

1Вкладка КАБЫ → отметь кабинеты
2Вкладка СЦЕНАРИИ → «+ из пресета…» → добавь 2-3
3Интервал 15 мин → ▶ СТАРТ
4Смотри ЖУРНАЛ — что бы сценарий сделал, если бы был боевым

🅑 Боевой режим

1Погоняй сценарий в observe день-два, сверь сработки
2Тумблер ARMED на конкретном сценарии → подтверди
3Теперь сработка = реальная пауза кампании/ада + запись в журнал
4⚙ → впиши TG-токен + chat_id — алерты прилетят даже когда не смотришь

Ограничения (честно)

⚠️ Вкладка = двигатель. Закрыл вкладку / браузер / ноут уснул — вотчдог стоит. Запинь вкладку, выключи Memory Saver для facebook.com. Это резервный контур, не замена серверному.
⚠️ Только FB-метрики. Трекерные данные (KT-инсталы/реги/депы) недоступны из браузера — реги/покупки здесь = события пикселя/CAPI, которые видит сам FB.
🔐 Безопасность: букмарклет работает только на твоей FB-сессии (cookies), никуда данные не шлёт (кроме Telegram, если сам включил алерты). Весь код в твоём браузере. Source: watchdog.js в этом же репо.
