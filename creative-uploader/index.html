Creative Uploader

Upload images to Facebook Ad Account directly from Ads Manager.
Drag & drop a folder — get image hashes instantly.

Install

1
Drag the button to your bookmarks bar
📤 Creative Uploader
↑ drag this to your bookmarks bar
2
Open Ads Manager inside Business Manager

Go to business.facebook.com → your ad account. Token and account ID are auto-detected from the page.

3
Click the bookmarklet

The uploader panel opens on the right side. Drop your images, hit Upload.

How it works

🔑
Token auto-detected — pulled from the page session, same as MetaCtrl PRO. No manual copy-paste.
🏦
Account ID from URL — reads act=… from the current Ads Manager page.
📁
Drag & drop folder — select all files at once (PNG / JPG). Uploads one by one to avoid rate limits.
🔢
Image hashes — after upload you get a JSON with name → hash pairs. Copy and paste into your campaign config.
🎬
Videos — coming soon. FB requires async encoding + status polling — will be added in v0.2.
