FB Launcher v0.13.0 — Bookmarklet

🚀 FB Launcher

Запуск кампаний в Facebook Ads Manager через Marketing API из CSV-файла. Обходит баги bulk-upload, поддерживает multi-adset (1×M×N), CBO/ABO, Special Ad Categories.

①
Перетащи кнопку ниже в свою панель закладок браузера (Bookmarks Bar).
🚀 FB Launcher
②
Открой business.facebook.com или adsmanager.facebook.com (логин обязателен).
③
Нажми закладку FB Launcher → справа откроется панель → загрузи CSV → выбери аккаунт → Launch.

Что умеет

  • CSV / TSV авто-распознавание — comma из самописных генераторов, tab из FB-экспорта
  • Multi-adset (1×M×N) — группирует строки по Ad Set Name, создаёт отдельный adset под каждую группу
  • CBO/ABO авто-detect — если есть Campaign Daily Budget → CBO, иначе ABO с per-adset budget
  • Special Ad Categories — Financial / Housing / Employment / Credit / Politics / Gambling
  • US states + countries — region-targeting через FB Targeting Search API + fallback на статичную таблицу
  • Image + Video ads — image_hash для статики, video_id с авто-thumbnail
  • URL tag transform — заменяет sub-параметры (sub2, sub3...) на ID аккаунта при заливе
  • Rate limit + retry — exponential backoff на FB rate-limit ошибки
  • PAUSED by default — все создаётся на паузе, review перед запуском

Формат CSV

Поддерживается стандартный FB Ads Manager export-формат (UTF-16 TSV) и custom CSV генераторы (UTF-8 comma).

Минимум обязательных колонок:

  • Campaign Name, Campaign Objective, Buying Type, Campaign Bid Strategy, Special Ad Categories
  • Ad Set Name, Ad Set Daily Budget (для ABO) или Campaign Daily Budget (для CBO)
  • Countries, Age Min, Age Max, Optimization Goal
  • Optimized Conversion Tracking Pixels, Optimized Event (или Pixel, Custom Event Type)
  • Ad Name, Title, Body, Link, Call to Action, URL Tags
  • Image Hash ИЛИ Video ID, Link Object ID (page_id, либо задать override в UI)

Воркфлоу

1Сгенерируй CSV в любом своем генераторе (или открой существующий FB-export)
2Загрузи готовые ассеты (картинки/видео) через FB Ads Manager Creative Uploader → получи hashes
3Подставь в CSV: реальный Pixel ID, Link Object ID (page_id), Link (tracker URL), Image Hash / Video ID
4Открой business.facebook.com → клик закладка FB Launcher → справа откроется панель
5В панели: upload CSV → preview покажет N adsets × M ads · CBO/ABO · SAC? → выбери target аккаунт
6Опционально: page_id override, URL tag режим, status (PAUSED/ACTIVE)
7Click 🚀 Launch → live log: ✓ campaign → ✓ adset 1/M → ✓ ad 1/N ...
8Все в PAUSED — review в Ads Manager, потом enable manually
⚠️ Безопасность: букмарклет работает только на твоей FB сессии (cookies), никуда не отправляет данные. Весь код выполняется в твоем браузере. Source: launcher.js в этом же репо.
