🚀 MetaLaunch PRO

Сборка и запуск рекламных кампаний в Facebook Ads Manager через Marketing API — прямо в панели или из CSV. Обходит баги bulk-upload: multi-adset (1×M×N), CBO/ABO, Special Ad Categories, geo-waterfall в один клик.

①
Перетащи кнопку ниже в свою панель закладок браузера (Bookmarks Bar).
🚀 MetaLaunch PRO
②
Открой business.facebook.com или adsmanager.facebook.com (логин обязателен).
③
Нажми закладку MetaLaunch PRO → справа откроется панель → собери кампанию (или загрузи CSV) → Launch.

Что умеет

  • Два входа — собери кампанию в панели с нуля или импортни готовый CSV / TSV (comma из генераторов, tab из FB-экспорта)
  • Multi-adset (1×M×N) — отдельный adset под группу, креативы раскидываются matrix'ей по адсетам
  • Geo-waterfall в клик — geo-кластерные пресеты (SafeX HOME, gambling EN-T1 / DACH / Nordic…) → по адсету на кластер
  • CBO / ABO + бюджет / бид — переключение режима, budget amount, bid strategy + cap прямо в панели
  • Targeting в панели — placement-пресеты (платформы + позиции), US states + countries через FB Targeting Search API, attribution window
  • Page / IG identity — выбор page из dropdown (все страницы аккаунта), page-only режим (FB-only, IG dropped)
  • Creative uploader в панели — drag&drop картинок + two-phase video (image_hash / video_id + авто-thumbnail)
  • Special Ad Categories — Financial / Housing / Employment / Credit / Politics / Gambling (DSA EU payor)
  • Пресеты конфигов — сохранить / загрузить настройки залива (localStorage)
  • URL tag transform — заменяет sub-параметры (sub2, sub3…) на ID аккаунта при заливе
  • Rate limit + retry — exponential backoff на FB rate-limit ошибки
  • PAUSED by default — всё создаётся на паузе, review перед запуском

Два способа собрать залив

🅐 В панели (без CSV)

1Клик MetaLaunch PRO на business.facebook.com → панель справа
2Собери: имя + маркеры, гео-кластеры, placements, CBO/ABO + бюджет, page / IG
3Залей креативы (drag&drop) → creative-distribution matrix по адсетам
4Preview → выбери target аккаунт → 🚀 Launch → live log
5Всё в PAUSED — review в Ads Manager, потом enable

🅑 Из CSV

1Сгенерируй CSV (свой генератор или FB-export)
2Залей ассеты через Creative Uploader → получи image hashes
3Подставь в CSV: Pixel ID, Link Object ID, Link, Image Hash / Video ID
4Панель → upload CSV → preview N adsets × M ads · CBO/ABO · SAC?
5Выбери target аккаунт → 🚀 Launch (всё в PAUSED)

Формат CSV (если льёшь из CSV)

Поддерживается стандартный FB Ads Manager export-формат (UTF-16 TSV) и custom CSV генераторы (UTF-8 comma).

Минимум обязательных колонок:

  • Campaign Name, Campaign Objective, Buying Type, Campaign Bid Strategy, Special Ad Categories
  • Ad Set Name, Ad Set Daily Budget (для ABO) или Campaign Daily Budget (для CBO)
  • Countries, Age Min, Age Max, Optimization Goal
  • Optimized Conversion Tracking Pixels, Optimized Event (или Pixel, Custom Event Type)
  • Ad Name, Title, Body, Link, Call to Action, URL Tags
  • Image Hash ИЛИ Video ID, Link Object ID (page_id, либо задать override в UI)
⚠️ Безопасность: букмарклет работает только на твоей FB сессии (cookies), никуда не отправляет данные. Весь код выполняется в твоём браузере. Source: launcher.js в этом же репо.
