Facebook Ads Toolkit

Набор bookmarklet-инструментов для работы в Facebook Ads Manager: авторулы, сборка и запуск кампаний, заливка креатива. Выбери инструмент — перейдёшь на его страницу с установкой и описанием.

MetaCtrl PRO

Авторулы · Аналитика

Панель управления в Ads Manager: генератор автоправил паузы/включения, column presets, аналитика и inspector кампаний.

  • Авторулы pause/unpause через FB Marketing API
  • Column presets — быстрые наборы колонок
  • Analytics + Inspector сущностей кампаний

MetaLaunch PRO

Сборка + запуск

Сборка и запуск кампаний в Ads Manager через Marketing API — прямо в панели или из CSV. Обходит баги bulk-upload.

  • Сборка кампании в панели или импорт CSV / TSV
  • Multi-adset 1×M×N, CBO/ABO, geo-waterfall в клик
  • Creative uploader, page/IG identity, SAC + DSA EU

MetaWatch PRO

Вотчдог · Сценарии

Вотчдог во вкладке Ads Manager — резервный контур мониторинга, независимый от серверной инфры. Сценарии по метрикам, тик по таймеру, журнал, TG-алерты.

  • Конструктор сценариев + готовые пресеты
  • OBSERVE / ARMED — от наблюдения до реальной паузы
  • Стата по кабам + журнал + Telegram из браузера

Creative Uploader

Хеши для CSV

Отдельный инструмент для заливки креатива в аккаунт и выдачи JSON-хешей. Результат идёт входом в FB Launcher / @fbcsv для сборки кампаний.

  • Batch-загрузка изображений в ad account
  • Копирование готового JSON с image hashes
  • Мост к FB Launcher и CSV-генератору
