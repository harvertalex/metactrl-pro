FB Switchboard v0.2.0 — Bookmarklet

🎛️ FB Switchboard

Кампании, адсеты и объявления в одной боковой панели Ads Manager. Смотришь статус, фильтруешь и щёлкаешь Пауза / Старт прямо через Graph API — без похода в UI кабинета.

①
Перетащи кнопку ниже в свою панель закладок браузера (Bookmarks Bar).
🎛️ FB Switchboard
②
Открой adsmanager.facebook.com на нужном кабинете (логин обязателен) и жми закладку — справа выедет панель. Повторный клик — закрыть.

Что умеет

  • Три вкладки: Кампании / Адсеты / Объявления — грузятся лениво, кэшируются
  • Фильтр по статусу: Активные · Актив+Пауза · Все + поиск по названию
  • Тумблер ⏸ Пауза / ▶ Старт у каждой строки — status=ACTIVE|PAUSED через API
  • Цветная точка = effective_status (зелёный / серый / красный=issues / жёлтый=review)
  • Дропдаун всех доступных кабинетов, текущий определяется из URL

Как это работает

1
Токен берётся со страницы Ads Manager (__accessToken) — ничего вводить не нужно, тот же способ что у MetaCtrl / MetaLaunch.
2
Кнопка смотрит на собственный status сущности, не на effective — поэтому не путается, когда адсет стоит из-за паузы кампании.
3
После переключения одна сущность ре-фетчится → показывает реальный статус. Пагинация с потолком 3000 строк на уровень.
🎛️ Standalone-инструмент: только чтение списков + переключение статуса. Массовых операций нет — только пер-строчный тумблер. Токен со страницы, ничего никуда не отправляется кроме Graph API.
